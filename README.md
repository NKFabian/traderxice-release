## Getting Started

### Installation

1. **Download the latest release** from the [Releases page](https://github.com/NKFabian/traderxice-release/releases)
   - **Windows**: `TraderXice Setup X.X.X.exe`
   - **macOS**: `TraderXice-X.X.X.dmg`
   - **Linux**: `TraderXice-X.X.X.AppImage`

2. **Install** by running the downloaded file
   - Windows: Follow the installer prompts
   - macOS: Drag TraderXice to Applications folder
   - Linux: Make AppImage executable and run it

3. **Launch TraderXice** and select your Obsidian vault folder

### First Session

1. Open TraderXice
2. Select or create an Obsidian vault
3. Fill in session details (type, strategy, pair, timeframe)
4. Click **Create Session**
5. Open your vault in Obsidian to start journaling
